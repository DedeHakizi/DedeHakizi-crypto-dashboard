# 🚀 DedeHakizi Crypto Dashboard

A real-time cryptocurrency portfolio tracker built with **React**, **Tailwind CSS**, and the **CoinGecko API**.

## 🛠️ Tech Stack
- **Frontend:** React 18, Vite
- **Styling:** Tailwind CSS
- **Data:** CoinGecko API (Live prices, market cap, 24h change)
- **State Management:** React Hooks (useState, useEffect)
- **Version Control:** Git & GitHub

## 📊 Features
- Live price updates for top cryptocurrencies (BTC, ETH, BCH, etc.)
- Portfolio value calculator in **EUR**
- 24-hour price change indicators (Green/Red)
- Dark mode UI for developer comfort
- Responsive design for mobile and desktop
