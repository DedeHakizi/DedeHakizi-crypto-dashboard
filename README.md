# 🚀 DedeHakizi Crypto Dashboard

A real-time cryptocurrency tracking and analytics dashboard built with modern web technologies.

## What's Inside

This project is your personal entry point into the crypto world—a dashboard to monitor, analyze, and track cryptocurrency trends at a glance.

### Stack

- **Languages:** JavaScript (78%), CSS (21%)
- **Frontend:** Vanilla JavaScript + HTML/CSS (lightweight, no bloat)
- **Purpose:** Real-time crypto price tracking, portfolio management, and market analysis
